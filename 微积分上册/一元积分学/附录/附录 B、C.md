
## 附录 B：希腊字母表

在数学、物理等学科中，希腊字母经常被用作各种符号。 以下列出了所有希腊字母的大写、小写以及英文/中文读音，方便查阅。

| 大写 | 小写   | 英文           | 中文读音 |
| ---- | ------ | -------------- | -------- |
| Α    | α      | Alpha          | 阿尔法   |
| Β    | β      | Beta           | 贝塔     |
| Γ    | γ      | Gamma          | 伽马     |
| Δ    | δ      | Delta          | 德尔塔   |
| Ε    | ε      | Epsilon        | 伊普西隆 |
| Ζ    | ζ      | Zeta           | 泽塔     |
| Η    | η      | Eta            | 伊塔     |
| Θ    | θ      | Theta          | 西塔     |
| Ι    | ι      | Iota           | 约塔     |
| Κ    | κ      | Kappa          | 卡帕     |
| Λ    | λ      | Lambda         | 兰姆达   |
| Μ    | μ      | Mu             | 米欧     |
| Ν    | ν      | Nu             | 纽       |
| Ξ    | ξ      | Xi             | 克西     |
| Ο    | ο      | Omicron        | 奥密克戎 |
| Π    | π      | Pi             | 派       |
| Ρ    | ρ      | Rho            | 柔       |
| Σ    | σ/ς   | Sigma          | 西格玛   |
| Τ    | τ      | Tau            | 陶       |
| Υ    | υ      | Upsilon        | 于普西隆 |
| Φ    | φ/ϕ   | Phi            | 斐       |
| Χ    | χ      | Chi            | 卡       |
| Ψ    | ψ      | Psi            | 普赛     |
| Ω    | ω      | Omega          | 欧米伽   |

**说明：**

*   在数学公式中，希腊字母的小写形式更为常用。
*   有些希腊字母的大写形式与拉丁字母相同，例如 A, B, E, H, I, K, M, N, O, P, T, X, Z。
*   $\sigma$ 在词尾时写成 ς。
*   $\phi$ 和 $\varphi$ 是 Phi 的两种不同写法，都可以使用。
